# OTA 公钥证书

## 📁 文件说明

此目录包含OTA更新验证所需的公钥：

- **`ed25519_public_key.pem`**: Ed25519公钥，用于验证OTA更新包的数字签名

## 🎯 用途

- **运行时验证**: 应用程序运行时使用此公钥验证下载的更新包
- **安全保障**: 确保更新包未被篡改，来源可信
- **自动发现**: OTA系统会自动查找此路径的公钥文件

## 🔒 安全说明

- ✅ **公钥安全**: 公钥可以公开，不需要特殊保护
- ✅ **运行时必需**: 删除此文件会导致OTA更新验证失败
- 🔐 **对应私钥**: 私钥位于 `build_system/certificates/` (仅构建时使用)

## 📋 技术细节

- **算法**: Ed25519 (椭圆曲线数字签名)
- **格式**: PEM (Privacy-Enhanced Mail)
- **大小**: 约113字节
- **用途**: 数字签名验证 (不用于加密)
